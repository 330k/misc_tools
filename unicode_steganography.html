Usage

unicodeSteganographer.encodeSteganography(text1, text2)

encodeSteganography returns a string which is embedded steganography of text2 into text1.

unicodeSteganographer.decodeSteganography(text)

decodeSteganography returns an array of string, 1st element is plain text, and 2nd element is hidden(decoded) text.

unicodeSteganographer.setUseChars

unicodeSteganographer.setUseChars sets the characters for steganography.
Default characters used for steganography are U+200C, U+200D, U+202C, and U+FEFF.
U+200B is deleted in Gmail when a mail is sent.
